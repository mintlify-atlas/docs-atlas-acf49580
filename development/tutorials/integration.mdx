This tutorial guides you through adding an integration that connects Zulip with a third-party service. We'll cover both **incoming webhooks** (service sends data to Zulip) and **bot integrations** (Zulip bot interacts with service).

## Integration Types

Zulip supports several integration patterns:

### Incoming Webhooks

Third-party service sends HTTP requests to Zulip:

- GitHub sends notifications on pushes/PRs
- Sentry sends error alerts  
- Jira sends issue updates

**When to use**: Service has outgoing webhook support

**Tutorial**: See [Webhook Tutorial](/development/tutorials/webhook)

### Bot Integrations

Zulip bot actively queries third-party service:

- Google Calendar bot checks for upcoming events
- RSS bot fetches new articles
- Weather bot queries weather API

**When to use**: Service has API but no webhooks, or need polling

**This tutorial focuses on bot integrations.**

### Native Integrations

Deep integration requiring server-side code:

- LDAP authentication
- Email mirroring
- Import from Slack/Mattermost

**When to use**: Core server functionality

## Bot Integration Overview

A bot integration:

1. Runs as a separate process
2. Uses Zulip API to send/receive messages
3. Queries third-party service API
4. Formats data and sends to Zulip

Example: RSS bot polls RSS feeds and posts new articles to Zulip.

## Prerequisites

- Zulip development environment set up
- Python knowledge
- Familiarity with the third-party service's API
- API credentials for the service (if required)

## Step 3: Implement the Bot

### Basic Bot Template

**`my_integration.py`**:

```python
from typing import Dict, Any
from zulip_bots.lib import BotHandler

class MyIntegrationHandler:
    """
    This bot integrates Zulip with MyService.
    
    It checks MyService for new items and posts them to Zulip.
    """
    
    def usage(self) -> str:
        return """This bot posts updates from MyService."""
    
    def initialize(self, bot_handler: BotHandler) -> None:
        """
        Called once when bot starts.
        Load config, initialize state, etc.
        """
        self.config_info = bot_handler.get_config_info('my_integration')
        self.api_key = self.config_info.get('api_key')
        self.check_interval = int(self.config_info.get('check_interval', 300))
        
        # Initialize any state
        self.last_check = None
    
    def handle_message(self, message: Dict[str, Any], bot_handler: BotHandler) -> None:
        """
        Called when bot receives a message.
        Respond to user commands.
        """
        content = message['content'].strip()
        
        if content.lower() == 'help':
            bot_handler.send_reply(message, self.usage())
            return
        
        if content.lower() == 'check now':
            self.check_for_updates(bot_handler)
            bot_handler.send_reply(message, "Checked for updates!")
            return
        
        bot_handler.send_reply(
            message,
            "I don't understand that command. Try 'help'."
        )
    
    def check_for_updates(self, bot_handler: BotHandler) -> None:
        """
        Query third-party service and post updates.
        """
        # Query the service API
        new_items = self.fetch_new_items()
        
        # Post each item to Zulip
        for item in new_items:
            message = self.format_message(item)
            bot_handler.send_message({
                'type': 'stream',
                'to': self.config_info.get('stream', 'general'),
                'topic': self.config_info.get('topic', 'MyService Updates'),
                'content': message,
            })
    
    def fetch_new_items(self) -> list:
        """
        Call third-party service API.
        """
        import requests
        
        response = requests.get(
            'https://api.myservice.com/items',
            headers={'Authorization': f'Bearer {self.api_key}'}
        )
        
        if response.status_code == 200:
            return response.json()['items']
        return []
    
    def format_message(self, item: Dict[str, Any]) -> str:
        """
        Format item data for Zulip.
        """
        return f"""**{item['title']}**

{item['description']}

[View details]({item['url']})"""

handler_class = MyIntegrationHandler
```

## Advanced Topics

### Polling Bots

For bots that need to poll periodically:

```python
import time
import threading

class PollingHandler:
    def initialize(self, bot_handler: BotHandler) -> None:
        self.config = bot_handler.get_config_info('my_bot')
        
        # Start polling thread
        self.polling = True
        thread = threading.Thread(target=self.poll_loop, args=(bot_handler,))
        thread.daemon = True
        thread.start()
    
    def poll_loop(self, bot_handler: BotHandler) -> None:
        while self.polling:
            try:
                self.check_for_updates(bot_handler)
            except Exception as e:
                print(f"Error polling: {e}")
            
            time.sleep(self.config.get('interval', 300))
```

### State Management

Store data between runs:

```python
def initialize(self, bot_handler: BotHandler) -> None:
    # Load last processed ID
    self.last_id = bot_handler.storage.get('last_id')
    if self.last_id is None:
        self.last_id = 0

def check_for_updates(self, bot_handler: BotHandler) -> None:
    items = self.fetch_items_since(self.last_id)
    
    for item in items:
        self.post_item(item, bot_handler)
        self.last_id = max(self.last_id, item['id'])
    
    # Save state
    bot_handler.storage.put('last_id', self.last_id)
```

### Error Handling

```python
import requests
from requests.exceptions import RequestException

def fetch_items(self) -> list:
    try:
        response = requests.get(
            self.api_url,
            headers={'Authorization': f'Bearer {self.api_key}'},
            timeout=10
        )
        response.raise_for_status()
        return response.json()['items']
        
    except RequestException as e:
        print(f"Error fetching items: {e}")
        return []
    except (KeyError, ValueError) as e:
        print(f"Error parsing response: {e}")
        return []
```
