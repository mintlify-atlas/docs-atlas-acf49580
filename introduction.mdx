## What is Zulip?

Zulip is the only modern team chat app that is designed for both live and asynchronous conversations. Fortune 500 companies, leading open source projects, and thousands of other organizations use Zulip every day.

Unlike other chat apps where messages are organized by who sent them, Zulip uses **topics** to organize messages by what they're about. This makes it easy to follow conversations, participate when convenient, and search for past discussions.

## Key Features

<AccordionGroup>
  <Accordion title="Topic-based Threading">
    Every conversation in Zulip happens in a stream (like a channel), tagged with a topic. This allows you to:
    - Follow multiple conversations in the same stream
    - Catch up on conversations you missed
    - Participate asynchronously without losing context
    - Search and reference past discussions easily
  </Accordion>

  <Accordion title="Powerful Message Formatting">
    Zulip supports rich message formatting including:
    - Full Markdown syntax with GFM extensions
    - Syntax-highlighted code blocks
    - LaTeX for mathematical expressions
    - Emoji reactions and custom emoji
    - Link previews and image previews
    - Polls and todo lists
  </Accordion>

  <Accordion title="Real-time Synchronization">
    All your messages, read state, and notifications sync instantly across:
    - Web application
    - Desktop apps (Windows, macOS, Linux)
    - Mobile apps (iOS and Android)
    - Terminal client
  </Accordion>

  <Accordion title="Extensive Integrations">
    Connect Zulip with your workflow through:
    - 100+ native webhook integrations (GitHub, Jira, Sentry, etc.)
    - Custom incoming webhooks
    - Outgoing webhooks for custom bot logic
    - Full REST API for building custom integrations
    - Zapier and IFTTT support
  </Accordion>

  <Accordion title="Enterprise-grade Authentication">
    Flexible authentication options including:
    - SAML SSO
    - LDAP/Active Directory
    - OAuth (Google, GitHub, GitLab, Apple)
    - Email/password with 2FA
    - Custom authentication backends
  </Accordion>

  <Accordion title="Self-hosting Options">
    Deploy Zulip your way:
    - Docker deployment
    - Native installation on Ubuntu/Debian
    - Pre-built images for Digital Ocean and Render
    - Or use Zulip Cloud for managed hosting
  </Accordion>
</AccordionGroup>

## Use Cases

### Open Source Projects

Zulip is the chat platform of choice for major open source communities including Rust, PureScript, and Wikimedia. The topic-based threading model makes it easy for newcomers to find relevant conversations and participate.

### Remote Teams

With support for asynchronous communication and powerful search, Zulip helps distributed teams stay connected across time zones without the pressure of real-time chat.

### Large Organizations

Zulip scales to thousands of users with features like guest access, permissions management, SAML SSO, and SCIM provisioning.

### Development Teams

Rich code formatting, GitHub integration, and API-first design make Zulip ideal for software development teams who need to discuss code and technical topics.
