## Endpoint

```
GET /api/v1/users/me/{stream_id}/topics
```

Get all topics in a channel.

## Path Parameters

<ParamField path="stream_id" type="integer" required>
  The ID of the channel.
</ParamField>

## Response

<ResponseField name="topics" type="array">
  Array of topic objects.
</ResponseField>

### Topic Object

<ResponseField name="name" type="string">
  Topic name.
</ResponseField>

<ResponseField name="max_id" type="integer">
  ID of the latest message in this topic.
</ResponseField>

## Examples

<CodeGroup>

```bash cURL
curl -X GET https://your-domain.zulipchat.com/api/v1/users/me/42/topics \
  -u email@example.com:API_KEY
```

```python Python
import zulip

client = zulip.Client(config_file="~/.zuliprc")

# Get topics in a channel
result = client.get_stream_topics(42)

for topic in result['topics']:
    print(f"Topic: {topic['name']} (latest message: {topic['max_id']})")
```

```javascript JavaScript
const result = await client.streams.topics.retrieve({
  stream_id: 42
});

console.log(result.topics);
```

</CodeGroup>
